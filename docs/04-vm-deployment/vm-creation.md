# 💻 VM Deployment

## 🎯 Objective
Deploy all virtual machines required for the lab.

---

## 🧩 Virtual Machines Created

- Ubuntu Server (Core + Wazuh)
- Ubuntu Desktop (CI/CD + Endpoint)
- Kali Linux (Attacker)
- Windows Endpoint

---

## ⚙️ Configuration

- CPU: 2 cores minimum
- RAM: 2–4GB
- Disk: 20GB+

---

## 🔐 Security Rationale

Separation of roles ensures:

- Isolation of attack surface
- Realistic enterprise simulation
- Controlled threat testing
