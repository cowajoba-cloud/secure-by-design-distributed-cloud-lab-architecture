# ⚠️ Threat Model Summary

## 🎯 Objective
Identify and categorize threats using **STRIDE methodology**.

---

## 🧠 STRIDE Analysis

| Threat | Description |
|------|------------|
| Spoofing | Identity impersonation |
| Tampering | Data modification |
| Repudiation | Lack of accountability |
| Info Disclosure | Data leakage |
| DoS | Service disruption |
| EoP | Privilege escalation |

---

## 🔐 CI/CD Threat Surface

Pipeline introduces risks such as:

- Malicious code injection
- Supply chain compromise
- Unauthorized artifact distribution
