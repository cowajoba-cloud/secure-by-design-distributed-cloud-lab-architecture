# 🌐 Network Design

## 🎯 Objective
Design a **segmented and controlled network environment**.

---

## 🧩 Design Approach

- Host-only network: `192.168.56.0/24`
- NAT adapter for internet access
- Isolation between attacker and target layers

---

## 🔐 Security Rationale

- Prevent lateral movement
- Enforce boundary controls
- Enable controlled attack simulation
