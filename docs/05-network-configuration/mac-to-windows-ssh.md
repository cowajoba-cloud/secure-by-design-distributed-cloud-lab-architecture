# 🔐 macOS to Windows SSH Tunnel

## 🎯 Objective
Enable secure remote access from macOS to Windows host.

## ⚙️ Command

bash : ssh -L 5602:192.168.56.100:443 user@windows-ip

---

## 🔐 Security Benefit
Encrypted tunnel
No direct exposure of services
Secure remote dashboard access
